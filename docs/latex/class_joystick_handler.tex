\subsection{Descripción detallada}
Genera una interfaz para poder recibir informacion de 1 joystick generico. 

~\newline
 A travez de la libreria de programas multimedia, S\+F\+M\+L, esta libreria obtiene los valores analogicos de los botones del joystick. ~\newline
 Proporciona un enumerador de botones que puede ser usado en otros programas ~\newline
 Y posee funciones que muestran si algun boton especifico esta presionado o acaba de ser soltado.
