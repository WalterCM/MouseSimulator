\subsection{Descripción detallada}
Genera una interfaz para poder controlar acciones de simulador de mouse. 

~\newline
 Esta clase usa funciones de las librerias de xorg xlib y xtest para generar acciones de mouse falso. ~\newline
 Esta clase puede\+:
\begin{DoxyItemize}
\item Mover el cursor a una posicion especifica de la pantalla
\item Mover el cursor una distancia especifica
\item Hacer clicks derechos
\item Hacer clicks izquierdos
\item Hacer click medio 
\end{DoxyItemize}
